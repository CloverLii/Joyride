
****************************** NOTE:  Completing Installation ******************************
********************************************************************************************

In order to complete the installation, install the NUnitRunners dependency in either your solution or test project:

1. Add the following to your packages.config file
<package id="NUnit.Runners" version="2.6.4" targetFramework="net45" />

The Joyride.Tools encapsulates the Nunit Runner and Specflow to provide helpful tools to run specs and generating the report.  

Enjoy!
